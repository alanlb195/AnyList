# Dev
1. Clonar el proyect
2. Copiar el ```env.template``` y renombrar a ```.env```
3. Ejecutar ```npm run dev```
4. Levantar imagen (docker desktop) ```docker compose up -d```
5. Levantar el backend ```npm run start:dev```
6. Visitar sitio ```localhost:3000/graphql```
